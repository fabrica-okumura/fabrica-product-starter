# Fabrica Product Starter

プロダクト新規作成用のスターターです。
Next.jsベースで、`fabrica-design-system` の利用を前提に構成しています。

## Setup

```bash
npm install
npm run dev
```

## Storybook

```bash
npm run storybook
```

## Design System

このスターターは `fabrica-design-system` を利用します。

### Private Registry を使う場合

`.npmrc` を有効化して `NPM_TOKEN` を設定してください。
